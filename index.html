Common CSS properties

Text Properties Table

Property Description Value
Color Sets the color of a text RGB, hex, keyword
Font-family Specifies the font family for text family-name, generic-family, inherit
Font-size Specifies the font size of text length, %, inherit
Font-style Specifies the font style for text normal, italic, oblique, initial, inherit
Text-align Aligns the text in an element left, right, center, justify
